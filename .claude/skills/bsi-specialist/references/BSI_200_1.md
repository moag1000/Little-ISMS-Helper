# BSI-Standard 200-1: Managementsysteme fuer Informationssicherheit (ISMS)

## Ueberblick

BSI-Standard 200-1 definiert allgemeine Anforderungen an ein ISMS und ist kompatibel mit ISO 27001:2022. Er beschreibt den Rahmen fuer die systematische Steuerung der Informationssicherheit.

## Kernelemente

### 1. Initiierung des Sicherheitsprozesses
- **Verantwortung der Leitungsebene**: Management muss IS-Prozess initiieren und steuern
- **Sicherheitsleitlinie**: Grundsatzdokument mit Stellenwert, Sicherheitszielen, Organisationsstruktur
- **IS-Organisation**: Rollen und Verantwortlichkeiten festlegen
  - Informationssicherheitsbeauftragter (ISB)
  - IS-Management-Team
  - IS-Koordination in Fachabteilungen

### 2. IS-Organisation

| Rolle | Verantwortung |
|-------|--------------|
| **Leitungsebene** | Gesamtverantwortung, Ressourcen, Leitlinie |
| **ISB** | Steuerung IS-Prozess, Berichterstattung, Koordination |
| **IS-Management-Team** | Operative Umsetzung, Massnahmenplanung |
| **Fachverantwortliche** | Umsetzung in ihrem Bereich |
| **Alle Mitarbeiter** | Einhaltung Richtlinien, Meldung von Vorfaellen |

### 3. Dokumentation

**Pflichtdokumente:**
- Sicherheitsleitlinie
- Sicherheitskonzept
- Risikoanalyse (bei erhoehtem Schutzbedarf)
- Realisierungsplan
- Notfallvorsorgekonzept

**Aufbewahrung:**
- Klassifizierung der Dokumente
- Versions- und Aenderungsmanagement
- Zugriffskontrolle auf IS-Dokumentation

### 4. Aufrechterhaltung und Verbesserung

**PDCA-Zyklus:**
- **Plan**: Sicherheitskonzept erstellen/aktualisieren
- **Do**: Massnahmen umsetzen, Awareness schaffen
- **Check**: Audits, IS-Revision, Kennzahlen pruefen
- **Act**: Korrekturmassnahmen, kontinuierliche Verbesserung

### 5. Kompatibilitaet ISO 27001

| BSI 200-1 | ISO 27001:2022 |
|-----------|----------------|
| Sicherheitsleitlinie | 5.2 Policy |
| IS-Organisation | 5.3 Organizational roles |
| Ressourcenmanagement | 7.1 Resources |
| Kompetenz & Awareness | 7.2, 7.3 Competence, Awareness |
| Dokumentation | 7.5 Documented information |
| IS-Revision | 9.2 Internal audit |
| Management-Bewertung | 9.3 Management review |
| Korrekturmassnahmen | 10.2 Corrective action |
